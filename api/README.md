# Decent Apartments - API Documentation

The static JSON API exposes renter article metadata for Decent Apartments. Endpoints are generated by Jekyll and require no authentication.

**Base URL:** `https://decent.apartments`

## Endpoints

### All Articles

**Endpoint:** `/api/guides.json`

Returns all renter articles with metadata such as title, URL, description, article stage, tags, estimated reading time, series data, image URL, and content length.

### Article Series

**Endpoint:** `/api/series.json`

Returns multi-part renter article series, such as apartment search essentials, lease basics, and rental red-flag guides.

### Topics and Tags

**Endpoint:** `/api/topics.json`

Returns article tags with related renter articles grouped by topic.

## Notes

The public language is “articles,” but the historical endpoint names still use `guides` to preserve URL compatibility.
